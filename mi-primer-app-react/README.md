# Trabajo Práctico Programación III

## ¿Para qué sirve el archivo package.json?

El archivo package.json contiene la información del proyecto, las dependencias utilizadas, los scripts de ejecución y la configuración necesaria para que la aplicación funcione correctamente. Permite instalar y administrar los paquetes del proyecto mediante npm.

## ¿Qué diferencia notaron entre ejecutar un archivo directamente con Node.js y levantar el servidor de React?

Al ejecutar un archivo con Node.js, el código se procesa directamente desde la terminal y el resultado se muestra en la consola.

Al levantar el servidor de React con Vite, se inicia una aplicación web que se visualiza en el navegador. Además, los cambios realizados en el código se reflejan automáticamente sin reiniciar el servidor gracias al Hot Module Replacement (HMR).
